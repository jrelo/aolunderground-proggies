FDO File Extensions
by Tau

Setting the FDO file type icon allows you to easily distinguish between them and other file types and setting FDO files to open in Notepad makes editing them easy to edit. I use the same icon that the AOL Database Viewer uses which makes it easy to remember. The icon looks like the following and it can be downloaded here.

Windows Association

In this step you will set the icon for all *.fdo files on your computer and make all FDO streams open in Notepad. These steps may vary on different Windows versions. These steps are for Windows 95.

  1. Open "My Computer"
  2. Click View | Folder Options...
  3. Click the File Types tab
  4. Click "New Type" To set the icon click "Change Icon". Hit "Browse" and navigate to where ever you download the FDO icon above.
  5. Set Description Type as Form Definition Order
  6. Set Associated Extension as fdo
  7. Set Content Type as text/plain
  8. Set Default Extension for Content Type as .fdo
  9. Now we will set it so when you open one of these in Explorer it will open the FDO stream in Notepad. Click "New..."
  10. For Action put %open
  11. Click "Browse" and navigate to WINDOWS\Notepad.exe
  12. Finally, click "Close"
